# Deploy on GitHub Pages

The Studio Privé Copacabana site will be deployed on GitHub Pages as a project page because it is a static Astro landing page with WhatsApp as the only conversion action. Vercel was considered for ease of deployment, but GitHub Pages better matches the cost constraint and avoids relying on Vercel Hobby for a commercial client site. A project page keeps this client site separate from the account's primary `usuario.github.io` site and requires Astro asset paths to account for the repository base path.
